Installation & Prérequis – AJVDNK

  1. Environnement requis
    • Un ordinateur sous Windows, Mac ou Linux
    • Un serveur local (WAMP, XAMPP, MAMP ou LAMP) avec PHP 7.4+ et MySQL/MariaDB
    • Un navigateur web moderne (Chrome, Firefox, Edge, ...)
  2. Installation du serveur local
    Téléchargez et installez WAMP (Windows), XAMPP (Windows/Mac/Linux) ou MAMP (Mac).
    Lancez le serveur (Apache + MySQL).
  3. Déploiement de l’application
    • Placez le dossier Projet AJVDNK dans le dossier www (WAMP) ou htdocs (XAMPP/MAMP).
    • Vérifiez que l’arborescence du projet est respectée.
  4. Création de la base de données
    • Ouvrez phpMyAdmin ou un terminal MySQL.
    • Importez le fichier db.sql fourni avec le projet.
    • Cela créera la base ajvdk_finance et toutes les tables nécessaires.
  5. Configuration de la connexion à la base
    • Ouvrez le fichier config/config.php.
    • Vérifiez/ajustez les identifiants MySQL (hôte, utilisateur, mot de passe, nom de la base) selon votre installation locale.
  6. Premier accès
    • Ouvrez votre navigateur et allez à l’adresse http://localhost/Projet%20AJVDNK/
    • Connectez-vous avec le compte trésorier créé dans la base (voir table users ou demandez à l’administrateur).
  7. Fonctionnalités principales
    • Gestion des membres, cotisations, dons, rapports, statistiques, profil trésorier.
    • Sécurité : authentification, gestion de session, modification du mot de passe.
  8. Problèmes fréquents
    • Erreur de connexion à la base : vérifier config.php et que MySQL est démarré.
    • Problème d’accès : vérifier les droits sur les dossiers/fichiers.
    • Mot de passe oublié : demander à l’administrateur de réinitialiser via la base.

Pour toute question, consultez l’aide utilisateur ou contactez le support.
